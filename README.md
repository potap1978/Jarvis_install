## 📖 Что это?

Этот скрипт позволяет развернуть **полноценного AI-ассистента в Telegram** с локальными нейросетями, мультимодальностью, голосовым общением и расширенными навыками.

**Всё, что нужно — запустить одну команду**, и через несколько минут у вас будет:

| 🎁 Что получаете |
|------------------|
| 🤖 Персональный ИИ-ассистент в Telegram |
| 🧠 44+ моделей на выбор (Qwen, Llama, DeepSeek, LLaVA, Grok) |
| 🎤 **Голосовое общение** — говорите, Джарвис отвечает голосом |
| 📸 Распознавание изображений |
| ⚡ Выполнение команд на сервере |
| 🔌 Подключение к Google, GitHub, API |
| 🕐 Автоматическая синхронизация времени |

---

## 🎯 Зачем это нужно?

Хотите **своего личного Джарвиса**, который:
- Работает на **вашем сервере**
- **Не отправляет данные в облако**
- Понимает **русский язык**
- **Видит фото**
- **Слышит и отвечает голосом**
- **Выполняет команды** на сервере

Этот скрипт автоматически настроит всё необходимое и даст вам удобные инструменты управления.

### 🤖 Основной бот (Telegram)

| Функция | Описание |
|---------|----------|
