# 🦞 JARVIS — Умный Telegram-ассистент с локальным ИИ

Скрипт создан для друзей Телеграм канала [Клуб «На связи»](https://t.me/na_sviazi66)

📖 Что это?
Этот скрипт позволяет развернуть полноценного AI-ассистента в Telegram с локальными нейросетями, мультимодальностью и расширенными навыками.

Всё, что нужно — запустить одну команду, и через несколько минут у вас будет:

🤖 Персональный ИИ-ассистент в Telegram

🧠 44+ моделей на выбор (Qwen, Llama, DeepSeek, LLaVA)

📸 Распознавание изображений и голоса

⚡ Выполнение команд на сервере

🔌 Подключение к Google, GitHub, API

🕐 Автоматическая синхронизация времени

🎯 Зачем это нужно?
Хотите своего личного Джарвиса, который работает на вашем сервере, не отправляет данные в облако, понимает русский язык, видит фото, слышит голос и может выполнять команды? Этот скрипт автоматически настроит всё необходимое и даст вам удобные инструменты управления.
